# 0004 — 3D view: open OSM extrusion now, photorealistic tiles parked (M10)

*Status: accepted · 2026-06-09*

## Context

M10 gives the operational map a real 3D view — terrain relief plus extruded
buildings. Terrain relief already worked (`setTerrain` drapes the raster basemap
over the Terrarium DEM). What was missing was building geometry: the app is
raster-basemap only, so there was nothing to extrude.

Two standing constraints (CLAUDE.md): **free / no-key only** (public-safe GitHub
Pages build) and **dependency-light**.

## Decision

Add buildings the open, keyless way: a MapLibre **`fill-extrusion`** layer over
**OpenFreeMap** vector tiles.

- **OpenFreeMap** (`https://tiles.openfreemap.org/planet`) serves OpenStreetMap
  vector tiles with **no API key, registration, or request limits**. Its
  OpenMapTiles-schema `building` source-layer carries `render_height` and
  `render_min_height` — exactly what `fill-extrusion` needs.
- The `buildings-3d` layer is declared up-front but hidden; the existing **3D
  toggle** (`set3D` → `setTerrain` + `setBuildings`) shows it and lifts it above
  the coverage raster. `minzoom: 14` keeps extrusion where it reads and the map
  performant. Neutral surface fill (`#3b4250`) on the always-dark map canvas.
- No new dependency (OpenFreeMap is a source URL); attribution is automatic via
  the vector source.

Buildings are **cosmetic/contextual** this milestone — they do **not** feed the
propagation model (clutter/landcover stays the M2 roadmap item).

## Parked (do NOT build now): photorealistic 3D tiles

Google **Photorealistic 3D Tiles** (Map Tiles API; OGC 3D Tiles via deck.gl
`Tile3DLayer` / Cesium / MapLibre+three.js) give Google-Earth city meshes but
require a **Google Cloud billing account + key** (pay-as-you-go). **Cesium OSM
Buildings** is a richer keyed middle tier. Both are key-gated/paid and so are
out of scope for the public-safe build.

Revisit if/when a key is acceptable (a private, coordinate-bearing build). The
open extrusion path above ships now and needs no key.

## Consequences

- **+** Real 3D buildings today, zero key, zero new dependency, OPSEC-safe.
- **+** Reuses the existing 3D toggle and DEM terrain; one control does both.
- **−** OSM building coverage/heights vary by region; some buildings lack height
  and extrude flat. Acceptable for contextual 3D.
- **−** Vector tiles add network requests at z≥14 (throttled by `minzoom`).
